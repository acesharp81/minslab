# 포트폴리오 실습 등록 방법

새 실습은 이 디렉터리 아래에 폴더 하나를 추가해 등록합니다.

```text
projects/
└── 04-my-practice/
    ├── project.json   # 화면에 표시할 정보
    ├── README.md      # 상세 학습 기록
    ├── main.py        # 대표 소스 코드
    └── assets/        # 이미지나 샘플 결과물(선택)
```

가장 가까운 기존 프로젝트의 `project.json`을 복사해 내용을 수정하면 됩니다.
`entry_file`에는 포트폴리오에서 보여줄 Python 또는 JavaScript 파일명을 적습니다.
`order` 값이 작을수록 좌측 메뉴 위에 표시됩니다.

## 외부 API 실습 주의사항

- API 키와 비밀번호는 프로젝트 파일에 적지 않습니다.
- 민감한 값은 `.env` 또는 서버 환경변수로 관리합니다.
- 실제 응답을 공유할 때 개인정보와 인증 토큰을 제거합니다.
- 큰 원본 데이터 대신 작은 샘플 데이터를 등록합니다.

파일을 추가한 뒤 아래 명령으로 서비스를 다시 시작합니다.

```bash
sudo systemctl restart myservice
```
